# PROJECT_CONTEXT: game69 String Art Studio

## 1. 背景
- 画像のような「2本の線分間を分割し、順番に結んで曲線風の模様を作る」ストリングアートを、ブラウザ上で再現する。
- ユーザーは任意の線分をキャンバス上に描き、2本を選択して自動生成された分割線をアニメーション表示できる体験を求める。

## 2. プロダクトゴール
- 数学的な線分補間を、直感的なUIで誰でも使える形にする。
- 描画した工程が視覚的に理解しやすく、教育・創作の両方で利用可能にする。

## 3. 想定ユーザー
- 幾何学模様や作図が好きな一般ユーザー。
- 授業やワークショップで可視化教材を作る教育者。
- SNS向けに短い制作過程動画を作りたいクリエイター。

## 4. 成果物の範囲（このスプリント）
- 要件定義書。
- 画面/操作フローを含むデザイン仕様書。
- ハーネス運用文書（Session Boot / Progress Board / PM Owner Channel）。

## 5. 非目標（このスプリントでは実施しない）
- 実コード実装。
- 永続化（クラウド保存、ログイン）。
- 複雑な3D描画。
