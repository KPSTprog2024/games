# Harness Meta Contents（ハーネス改善用メタコンテンツ）

このフォルダは、`harness/` 本体とは分離して、
**ハーネスの評価・改善・更新方針**を扱う。

- `harness/`: 実運用で毎回使うハーネス本体
- `harness_meta/`: ハーネス改善に関するメタ文書

## 読む順番

1. `20_HARNESS_UPDATE_POLICY.md`（更新方針・更新手順）
2. `10_HARNESS_REVIEW_AND_CHANGE_PROPOSAL_2026-04-18.md`（評価・改善提案の具体例）
3. `30_HARNESS_V2_RESTRUCTURE_2026-04-18.md`（v2構造変更の実施記録）

## 運用ルール

- メタ文書は実運用テンプレートに混在させない
- ハーネス本体を更新する際は、必ず `20_HARNESS_UPDATE_POLICY.md` の手順に従う
- 更新後は、変更理由と影響範囲を本フォルダに追記する
