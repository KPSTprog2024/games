Color and Drawing

This section explains the appearance controls:

  • Color Mode – choose between solid color or gradients.
  • Hue Base / Span – define gradient hues.
  • Saturation and Lightness – adjust HSL values.
  • Opacity – overall transparency of lines.
  • Line Width – thickness of the strings.
  • Glow – adds a subtle blur for a glowing effect.
