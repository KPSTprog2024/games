# game67 - Cat Tracker 4x3

4行3列のグリッドで猫の移動を追跡し、最後の位置を答える記憶ゲームです。

## 仕様
- グリッド: 4x3
- 1ラウンドごとの移動回数: `2 + round`
- 猫表示時間: 600msから開始し、ラウンドごとに前ラウンド比80%へ短縮（10ms下限）
- ラウンド内のテンポ: 猫の表示時間と非表示時間は同じ値を使い、1表示ごとに90%へ短縮（10ms下限）
- 表示時間の計算は小数第3位以下を切り捨て（小数第2位まで）
- 全14ラウンド

## 実行方法
`game67/index.html` をブラウザで開いてください。

## 操作
1. 「開始」ボタンでラウンド1を開始。
2. 猫の移動を目で追う。
3. 最後にいたマス番号をクリック。
4. 「次のラウンド」で継続、または「リセット」で再開始。
