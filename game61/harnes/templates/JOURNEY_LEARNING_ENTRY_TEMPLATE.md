# JOURNEY_LEARNING_ENTRY_TEMPLATE.md

`learning/AGENT_GROWTH_LEDGER.md` へ追記するための雛形。

```md
## ENTRY-XXXX
- Created-At: YYYY-MM-DD
- Journey-ID:
- Context:
- Reusable Insight:
  -
- Reasoning Pivot:
  -
- Do:
  -
- Don’t:
  -
- Evidence:
  -
- Reuse Plan:
  -
- Confidence:
  - low / medium / high
```

## 記入ガイド

- Reusable Insight: 他案件でも使える抽象知にする
- Reasoning Pivot: 判断が変わった瞬間と理由を書く
- Do / Don’t: 次回の実行規範に落とす
- Evidence: 参照文書・ログ・検証結果を添える
