# game63 出力マップ

このフォルダは、`GAME_ESSENCE.md` をもとに「5歳児の暗記（覚える・思い出す）能力を10歳相当に底上げする」ための設計議論を、後続の実装・企画更新に接続できる形式で整理した成果物です。

## 追加された運用ファイル（2026-04-15）

- `../PROJECT_MANAGER.md`
  - PMが毎回同じ品質で進行するための運用書
- `../NEXT_PROMPT_LOG.md`
  - 追記型の次回実行プロンプト管理台帳（LATESTポインタ付き）
- `07_SUPER_UPGRADE_MASTER_PLAN.md`
  - 圧倒的バージョンアップに向けた統合ロードマップ

## 収録物

1. `01_FACILITATION_FRAME.md`
   - ディレクターとしての進行設計（目的、評価軸、議論手順、合意ルール）
2. `02_PERSONA_PANEL_DISCUSSION.md`
   - 複数ペルソナによる論点整理済みディスカッションログ
3. `03_THREE_GAME_CONCEPTS.md`
   - 3案のゲーム体験（狙い、ループ、難易度、実装注意）
4. `04_UPDATE_BLUEPRINT_TEMPLATE.md`
   - 議論結果からアップデート版青写真へ落とすためのテンプレート
5. `05_REASONING_LOG_DESIGN.md`
   - 後追い可能性を担保する「思考ログ」の設計（保存粒度・項目・運用）
6. `06_REASONING_LOG_SAMPLE.md`
   - 今回案件の要約推論ログ（意思決定の痕跡）
7. `07_SUPER_UPGRADE_MASTER_PLAN.md`
   - 3案を統合した段階導入ロードマップ（Sprint 1〜3）

## 利用順序（推奨）

1. `../NEXT_PROMPT_LOG.md` の LATEST を確認
2. `../PROJECT_MANAGER.md` で実行ルールを確認
3. `07` で全体ロードマップ確認
4. `01`〜`03` で論点と採用案を確認
5. `04` で仕様化、`05`/`06` で判断ログを運用
